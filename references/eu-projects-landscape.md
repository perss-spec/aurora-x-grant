# EU-Funded Projects Landscape for AURORA X Positioning

## Key Related Projects

| Project | Programme | Budget | Focus | AURORA X Differentiation |
|---|---|---|---|---|
| FITGEN | H2020 | EUR 5.84M | Integrated e-axle (auto) 6 kW/kg | Aviation focus, >5 kW/kg, two-phase cooling |
| HiEFFICIENT | H2020 | EUR 41.85M | WBG modules for automotive | Monolithic motor-inverter integration for aviation |
| ASuMED | H2020 | EUR 4.78M | Superconducting motor 20 kW/kg | No cryogenics needed, practical for UAV |
| QualE-fly | EIC Trans. | EUR 2.5M | Certifiable drive for regional aircraft | UAV/eVTOL segment, different scale |
| IMOTHEP | H2020 | EUR 18.68M | HEP system analysis | Actual hardware, not analysis |
| HE-ART | Clean Aviation | EUR 44.1M | Hybrid regional propulsion | 10-50 kW class vs MW class |
| HECATE | Clean Aviation | EUR 34.2M | >500 kW distribution | Different power range |
| ALTOM | H2020 SME | EUR 71K | Two-phase cooling feasibility | Full integration into motor |

## Unfilled Gap

No EU project addresses: **Integrated electric drive (motor + WBG inverter + two-phase cooling) in 10-100 kW class for UAV/eVTOL with >5 kW/kg system power density.**
