# AURORA X Grant Proposal

## Project
European grant proposal (EIC Transition) for AURORA X — Smart Integrated Coaxial E-Propulsion Core.

## Structure
- `doc/` — Final assembled proposal document (Markdown + PDF)
- `src/sections/` — Individual sections of Part B
- `src/figures/` — Diagrams, charts, infographics (SVG/PNG)
- `src/tables/` — Data tables
- `src/calculations/` — Engineering calculations (Python notebooks)
- `templates/` — EU grant templates
- `references/` — Bibliography and source materials
- `assets/` — Raw images, logos

## Conventions
- Document language: English (EU grant requirement)
- Technical units: SI
- Currency: EUR
- All calculations must be reproducible (Python scripts/notebooks)
